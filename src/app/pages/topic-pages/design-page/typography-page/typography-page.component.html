Guidance and Usage

The importance of typography

A visual hierarchy of text enables a user to navigate a website or software application with ease. When a user sees consistent use of typography throughout a single digitial experience, it provides a sense of comfort on what actions that user should take.

Text alignment

While right-aligned, centered, and justified text have their place, most websites benefit from a consistent use of left-aligned text. Justified text, common in print, does not yet display well enough in a web browser to be considered a best practice.

Line Length

Most lines of text should be 45-90 characters. Somewhere between 45 and 90 characters per line is broadly considered to be a readable line length, with 66 characters considered a comfortable target for long texts.

Large text can have a shorter measure. Since larger text (<h1>, <h2>) takes up more screen real estate, it can make sense to assign it a relatively small measure.
