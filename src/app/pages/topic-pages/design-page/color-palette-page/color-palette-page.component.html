Guidance and Usage

The importance of color

The consistent use of color is curcial to maintaining the EPAS brand. Color helps users quickly identify what content they need to focus on and how to move through pages quickly.

Primary vs. Secondary

The color palette is compromised of a primary and secondary set of colors. The primary set is often seen the most often and therefore is a conservative set of colors. The secondary palette consists of brighter colors where specific content is emphasized.

Line Length

Most lines of text should be 45-90 characters. Somewhere between 45 and 90 characters per line is broadly considered to be a readable line length, with 66 characters considered a comfortable target for long texts.

Large text can have a shorter measure. Since larger text (<h1>, <h2>) takes up more screen real estate, it can make sense to assign it a relatively small measure.
