Icons

Icons communicate meaning in a graphical user interface.

Purpose of Icons

Icons are visual image like depictions of an idea, object, or actions. Icon usage should be accompanied by a label which ensures 508 compliance. The use of icons also allows users to quickly glance at an icon and understand the purpose the accompined html element services.

Additionally, EPAS uses icons along with color to depict areas (Travel, Security, etc.) as well as actions a user can take.

Font Awesome

EPAS applications use an icons library known as Font Awesome. Font Awesome is a font and icon toolkit based on CSS and LESS. Using Font Awesome, the unit has access to hundreds of icons and animations for free.
