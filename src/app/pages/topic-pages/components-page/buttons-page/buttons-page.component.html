Buttons

Use buttons to draw attention to important actions. Buttons allow users to take an action.

Types

There are four buttons that can be used in the UI. Each are listed below.

Button text

Strong Primary action button. Used to drawl users attention.
Strong with Icon Primary action button. Used to drawl users attention. Includes icon for additional information. Should be used sparingly.
Standard Secondary action button. Typically is not a critical action to complete flow of case.
Standard with Icon Secondary action button. Typically is not a critical action to complete flow of case. Includes icon for additional information. Should be used sparingly.
